# Noqta · Portfolio

Marketing + screenshots + public docs for **Noqta**, the offline-first multi-vertical POS for MENA small businesses.

This repo is also the source for the public landing site at `noqta.github.io` (free via GitHub Pages). Switch to a custom domain once the first client signs.

## Contents (to be built during Phase A)

- `/landing` — one-page marketing site (static, Tailwind, Arabic + English)
- `/screenshots` — cashier, back-office, KOT, receipt, dashboard
- `/case-studies` — client #1, #2, #3 stories as they go live
- `/pricing.md` — starter / pro / enterprise tiers
- `/integrations.md` — Talabat, Paymob, ETA, Instapay, printers, scales

## Why this exists separately from the code repo

Public repos get unlimited GitHub Actions minutes and free GitHub Pages hosting. Code stays private in `noqta/noqta` until launch; this stays public.
